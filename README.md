# Gerenciador de Tarefas em Python

Projeto acadêmico desenvolvido para a disciplina Linguagem e Técnica de Programação.

## Descrição
Este projeto consiste em um gerenciador de tarefas executado no terminal, permitindo ao usuário organizar suas atividades de forma simples e eficiente.

## Funcionalidades
- Adicionar tarefas
- Visualizar a lista de tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Pesquisar tarefas por palavras-chave

## Tecnologias utilizadas
- Python 3

## Como executar
1. Abra a pasta do projeto
2. No terminal, execute:
   ```bash
   python Gerenciador Tarefas.py
